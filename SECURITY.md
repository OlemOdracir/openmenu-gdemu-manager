# Security Policy

## Reporting Issues

Please report security-sensitive issues privately by email:

openmenu.gdemu.manager@gmail.com

Do not open a public issue for private credentials, unsafe file-writing behavior, or data-loss reports that include personal paths/logs.

## Sensitive Data

Do not attach ROMs, BIOS files, SD card backups, private API credentials, logs, or full local configuration files to public issues.

The app should not ship with third-party API credentials. Optional online providers must be configured locally by the user or routed through a project-controlled service that keeps secrets server-side.
