# Screenshots

Public screenshots used by the README and release notes.

Current files:

- `ogm-logo.png`
- `main-window.png`
- `online-sources.png`

Recommended screenshots for the public beta:

- main window after scanning a safe demo SD or sanitized backup;
- setup/diagnostic wizard showing a compatible SD;
- apply changes dialog with covers/additions/deletions;
- cover search/manual correction dialog;
- online sources settings.

Before committing screenshots, make sure they do not show credentials, private paths, personal files, ROM collections, BIOS files or other sensitive data.

Use a sanitized demo SD when possible. Avoid showing a full personal game list unless the titles are intentionally public in the release material.
